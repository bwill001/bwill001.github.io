Artifact 3: Parsing Delta Flight Data

The purpose of this group project was to visualize flights in an animation using real data provided by Delta. Delta had made available a file with all flights scheduled for a certain time period. This data was provided in a PDF, which I converted to a single Excel sheet and CSV file with only relevant information kept.

In the project, I learned more about how to wrangle data. When intially trying to convert data from the PDF, many things were not transcribed correctly, or there would be a lot of NULL data. I also renamed or rewrote specific entries to better suit our needs in the project.

Next time, I would utiliza a database for quicker loading and ability to easily query data.
