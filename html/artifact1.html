Artifact 1: Visualizing Georgia COVID-19 Data

The purpose of this project is to visualize data from the Georgia Department of Health regarding COVID-19 and to discover additional insights that are not explicitly recorded.

For me, this was also an opportunity to familiarize myself with more Python tools and frameworks, including Beautiful Soup, Selenium, and Plotly. I learned more about how to parse data from pure HTML, and ways to interactively visualize data in web applications.

Next time, I would define more methods to decrease repetitive tasks. I ended up using some of the same operations to transform all the data sets, which could have been made into functions.

Please scroll to view code and output related to the analysis portion of this project. Note that the visualizations are interactive.
